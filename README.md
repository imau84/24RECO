# 24reco.com

Platforma de rapoarte bazate pe date publice din Romania.

## Sectoare
- Auto (inmatriculari vehicule)
- Transport
- Imobiliare
- Institutii Publice

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts

## Rulare locala

```bash
npm install
npm run dev
```

Deschide [http://localhost:3000](http://localhost:3000)

## Deploy

Conectat la Vercel via GitHub. Orice push pe `main` declanseaza un deploy automat.
